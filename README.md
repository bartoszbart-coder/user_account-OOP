#UserAccount Login System

A simple Object-Oriented Python login system demonstrating encapsulation, validation, and account locking logic.

##Features

- Username validation
- Password validation (minimum 8 characters + at least 1 digit)
- Failed login attempt tracking
- Account lock after configurable number of failed attempts
- Read-only password property (cannot be accessed directly)
- Login and logout functionality

##Technologies Used

- Python 3
- OOP principles (Encapsulation, Properties, State Management)
